Bringing Project Management to Life

Interviews with Industry

How to use these materials

A series of filmed interviews has been undertaken with professionals in the software industry. These interviews have been structured around a set of themes. Each of these themes has been taken, and a digital video made of each. This project website is concerned with making these videos available.

Viewing the Videos

Each video can be viewed directly from your web browser.  To watch full-screen, simply click the full screen icon.  All the videos can be downloaded – see the links below the film viewer.  An index to the full set of videos is here.

Using for Teaching

Download a copy of the video, and incorporate into a PowerPoint or KeyNote presentation.
